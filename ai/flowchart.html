AI Minor Syllabus Flowchart

Mathematics for Data Science

  • Vectors, Matrices, Eigenvalues, Eigenvectors
  • Derivatives, Gradients, Chain Rule, Optimization
  • Conditional Probability, Bayes’ Theorem, Distributions
  • Set Theory & Logic

Statistics for Data Science

  • Descriptive Stats: Mean, Median, Mode, Variance
  • Inferential Stats: P-test, Z-test, T-test
  • Confidence Intervals, Histograms

Python Programming

  • Basics, Data Types, Control Structures
  • Functions, Error Handling, File Operations
  • Lists, Tuples, Sets, Dictionaries
  • Sorting Algorithms & Efficiency

Machine Learning

  • Regression, Polynomial Regression
  • Dimensionality Reduction, PageRank
  • Perceptron, Neural Networks, SVM
  • Supervised & Unsupervised Learning
  • Evaluation Metrics, K-Means Clustering

Deep Learning

  • CNN: Convolution, Pooling
  • Activation Functions: Sigmoid, ReLU, etc.
  • Architectures: LeNet, AlexNet, VGG, ImageNet

Natural Language Processing

  • Sequence Models: AR Models, Markov Models
  • Tokenization, Vocabulary, n-grams, Perplexity
  • RNNs, GRUs, LSTMs, BPTT
  • Word Embeddings, GloVe

Transformers & LLMs

  • Transformers, Bi-RNNs, Attention Mechanisms
  • Prompt Engineering, LLM Training
  • Applications, Evaluation & Efficiency
  • RAG: Building & Challenges

Semi-Supervised Learning

  • Ladder Networks, Π-Models
  • Self-Training, Co-Training
  • VAEs, GNNs, Label Propagation

Reinforcement Learning

  • Multi-Armed Bandit Problem
  • Markov Decision Processes
  • Dynamic Programming, Monte Carlo
  • Temporal Difference & Deep RL
